# Installation and Running
1. Clone the repo
   
   https://github.com/FredrikMohag/workflow-CA.git

3. Install the dependencies

   npm install

4. Run build/SASS

    nmp run start

   npm build

5. Check format
 
   npm run format

6. Check Lint
7. 
   npm runt lint
